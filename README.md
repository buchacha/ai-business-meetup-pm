# AI для продакта — лендинг митапа AI Business SPb

Интерактивная одностраничная страница для митапа по применению ИИ в продуктовой разработке. Содержит промпты, кейсы и ссылки на инструменты.

## Структура страницы

1. **Hero** — заголовок, ценностное предложение, блок «Для кого»
2. **Инструменты AI** — 6 карточек (DeepSeek, NotebookLM, Claude, Mermaid, Manus, Kimi)
3. **Практические кейсы** — 6 кейсов с попапами и копируемыми промптами
4. **Об авторе** — Егор Красильников, портфолио мероприятий, фотогалерея
5. **Контакт** — 2 карточки захвата лидов
6. **Футер** — ссылки на Telegram, AllSee.team, AI Business SPb

## Файлы

```
index.html        — вся страница (HTML + CSS + JS в одном файле)
images/           — фото и логотипы (добавляются вручную, см. images/README.md)
```

## Добавление медиафайлов

Все медиафайлы хранятся в папке `images/`. Страница корректно отображается без них (плейсхолдеры), но для полного вида нужно добавить файлы согласно `images/README.md`.

## Технологии

- Чистый HTML/CSS/JS, без фреймворков и сборщиков
- Шрифты: Russo One (заголовки), Inter (текст) — Google Fonts
- Цветовая схема: фон `#F4F7F6`, акцент `#422293` / `#5B2FD4`
